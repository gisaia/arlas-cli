
## Prerequisite

- python 3.10
- pip

If you manage your own ARLAS stack, you will also need:
- [ARLAS](https://github.com/gisaia/ARLAS-Exploration-stack)

## Install

<!-- termynal -->
```shell
> pip install arlas_cli
```

In a new terminal, you should be able to run it:

<!-- termynal -->
```shell
> arlas_cli --version
X.X.X
```
